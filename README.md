# Algorandprojectcertificate
# Algorand Course Project

This repository holds my Algorand blockchain project for the Algorand Developer Course (Assignment 1). The project will build a simple decentralized raffle dApp where users buy tickets with ALGO and winners are selected randomly via smart contracts.

## About
- **Author**: [gracyjula]
- **Course**: Algorand Developer Course - Assignment 1
- **Goal**: Demonstrate Algorand smart contracts for raffle creation, ticket purchases, and on-chain winner selection.
- **Technologies**: 
  - Algorand SDK / AlgoKit
  - Python / PyTeal for smart contracts
  - JavaScript for frontend (if planned)
  - Pera Wallet integration

## Current Status
- Initial repository setup complete with project plan.
- Planned features:
  - Smart contract for raffle logic (ticket sales, random draw).
  - Frontend UI for creating raffles and buying tickets.
  - Testnet deployment and transaction testing.
  - Wallet connection and ASA (raffle token) handling.

## Quick Start (To be updated)
1. Clone the repo: `git clone https://github.com/[gracyjula]/[Algorandprojectcertificate]`
2. Install dependencies: `pip install algokit py-algorand-sdk` (or `npm install` for JS).
3. Local dev: `algokit localnet start`
4. Deploy: `algokit project deploy`

## Roadmap
- **Week 1**: Install AlgoKit, write basic smart contract for ticket opt-in.
- **Week 2**: Add raffle creation and random selection logic.
- **Week 3**: Build simple JS frontend, test on testnet.
- **Future**: Deploy to mainnet, add UI improvements.
